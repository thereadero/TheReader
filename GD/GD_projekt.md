# GD Project

## Overview
GD is a simple Geometry Dash-inspired game built using Python and the Pygame library. The game features a controllable cube that navigates through obstacles in a side-scrolling environment. This project is currently under revision and development.

## Features
- **Player Character**: A green cube that can be controlled by the player.
- **User Interface**: Start button to initiate gameplay.
- **Game Mechanics**: Basic movement and collision detection (in development).
- **Window Settings**: Fixed window size of 800x500 pixels.

## Requirements
- Python 3.x
- Pygame library (`pip install pygame`)

## How to Run
1. Ensure Python and Pygame are installed.
2. Navigate to the GD directory.
3. Run the game with: `python GD.py`

## Project Structure
- `GD.py`: Main game script containing the game logic, classes (Button, Cube), and Pygame initialization.
- `GD_projekt.md`: This project documentation file.
